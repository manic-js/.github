# Manic JS

Manic JS is a Bun-first ecosystem for building and deploying high-performance React applications.

The fastest framework for React.

## Repositories

- Core framework: [manic-js/manic](https://github.com/manic-js/manic)
- App scaffolding: [manic-js/create-manic](https://github.com/manic-js/create-manic)
- Providers: [manic-js/providers](https://github.com/manic-js/providers)
- TUI utilities: [manic-js/tui](https://github.com/manic-js/tui)
